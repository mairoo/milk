reCAPTCHA v3 검증 테스트

사용법:
1. reCAPTCHA Site Key를 입력하세요
2. 최소 점수를 설정하세요 (기본값: 0.5)
3. "reCAPTCHA v3 검증" 버튼을 클릭하세요
4. reCAPTCHA v3이 자동으로 실행되어 토큰과 점수가 생성됩니다
5. 백엔드 API 응답을 확인하세요

테스트 엔드포인트: /open/recaptcha/v3/verify
reCAPTCHA v3 특징:
• 사용자 상호작용 없이 백그라운드에서 실행
• 0.0(봇) ~ 1.0(인간) 점수 반환
• 점수 기준으로 추가 검증 결정 가능
• 'verify_test' 액션으로 토큰 생성
점수가 높을수록 인간일 가능성이 높습니다 (기본값: 0.5, 비워두면 서버 기본값 사용)
